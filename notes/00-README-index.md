# Angular Master Notes — Index

Interview-oriented, production-relevant Angular reference notes, split into focused files so each topic is easy to navigate and revise independently.

## How this set is organized

- **Major topics** (Components, Directives, Pipes, Dependency Injection, Routing, Forms, HTTP Client, RxJS, Signals, Auth/JWT, Performance) get the **full 12-part treatment**: What/Why/When/How, Syntax, Examples, Pros/Cons, Interview Questions, Best Practices, Common Mistakes, Comparison Tables.
- **Supporting topics** (Fundamentals, CLI, TypeScript, Templates, State Management, Security, Testing) get a **condensed format** — definition, how it works, key syntax, example, best practices, and a short interview Q&A — since these are typically reference/setup material rather than deep conceptual topics.
